Conceitos de POO

Objeto: Um objeto é uma instância de uma classe e representa um elemento da vida real com características e comportamentos. Por exemplo, um objeto "carro" pode ter atributos como cor, modelo e marca, e métodos para acelerar, frear e virar.

Classe: Uma classe é um modelo ou uma planta para criar objetos. Ela define os atributos e métodos que os objetos dessa classe terão. Por exemplo, a classe "Carro" pode definir que todos os carros têm atributos como cor, modelo e marca, bem como métodos para operações relacionadas a carros.

Encapsulamento: É o conceito de esconder os detalhes internos de uma classe e expor apenas as interfaces públicas. Isso ajuda a manter a integridade dos dados e facilita a manutenção do código.

Herança: Herança é um mecanismo que permite que uma classe herde atributos e métodos de outra classe. Isso promove a reutilização de código e cria hierarquias de classes.

Polimorfismo: Polimorfismo permite que objetos de classes diferentes possam ser tratados de forma uniforme, simplificando a interação com objetos de classes diversas.

Abstração: A abstração é o processo de simplificar complexidades criando modelos que representam entidades do mundo real. Isso envolve identificar as características essenciais de um objeto e ignorar os detalhes não essenciais.
